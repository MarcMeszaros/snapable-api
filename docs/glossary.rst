Glossary
========

.. glossary::

    API
        `API <https://en.wikipedia.org/wiki/Application_programming_interface>`_ (Application Programming Interface) used by developers to intereact with Snapable. The Snapable API follows a `REST <https://en.wikipedia.org/wiki/Representational_state_transfer>`_ architecture design.

    JSON
        *A text-based open standard designed for human-readable data interchange.* [`Wikipedia <http://en.wikipedia.org/wiki/JSON>`_]

    nonce
        A `cryptographic nonce <http://en.wikipedia.org/wiki/Cryptographic_nonce>`_ used when signing API requests.

    resource
        In terms of a REST-style architecture, a *resource* is a collection of similar and related data.
